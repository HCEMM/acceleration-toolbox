## GitHub

### Version Control

In this workshop, we will use GitHub to collaborate on building a bioinformatics pipeline. GitHub is a platform for version control, which allows multiple people to work on the same project, track changes, and manage contributions efficiently.

### Before the Workshop

Please make sure you:

- Register for a GitHub account: https://github.com/
- Familiarize yourself with the interface (optional but helpful)
- Access the workshop repository (link will be provided)

### Getting Started

1. **Fork the repository**
   - Click the “Fork” button on the top right of the repository page
   - This creates your own copy of the project

2. **Clone the repository**
   - Copy the URL of your fork
   - In your terminal, run:
     ```bash
     git clone <your-fork-url>
     ```
   - Navigate into the project directory:
     ```bash
     cd <repository-name>
     ```

3. **Make a change**
   - Open the provided test file
   - Modify its contents (e.g., add your name or a short message)

4. **Commit your changes**
   ```bash
   git add <filename>
   git commit -m "Add my first change"
   ```
